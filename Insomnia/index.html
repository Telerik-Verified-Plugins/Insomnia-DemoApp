Insomnia Plugin Demo

The Insomnia plugin allows you to prevent the device screen from dimming (and locking the device) while the user is not touching the screen. You're in control of when your app allows the screen to dim again.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android/WinPhone - App package.

Stay awake

You can run this directly after 'deviceready' has fired to never allow the screen to dim, or you can trigger it at anytime during the lifecycle of your app.

Note that the time before the screen dims can be controlled in the settings of your device - you may want to set it as low as possible to test this plugin.

Allow sleep again

You can allow the device to dim the screen (and eventually lock the device) by calling a second plugin function.
